Wifi-Bomb
===================

The **Wifi-Bomb** project disrupts the connection of connected device to a router. The application requires to know the MAC address of the connected device.

----------

Requirement
-------------
Wemos D1 Mini V2 (ESP8266) [here][1]
ON/OFF Switch 
2*battery AAA LR03
